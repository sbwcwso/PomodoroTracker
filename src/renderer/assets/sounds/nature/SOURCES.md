# Nature sound sources

These loopable OGG files come from the open `Muges/ambientsounds` collection:

- Repository: https://github.com/Muges/ambientsounds
- `fireplace.ogg`: “Fireplace” by inchadney — CC0
- `heavy-rain.ogg`: “Heavy Rain” by D W — CC BY
- `forest-rain.ogg`: “Forest Rain” by Corsica_S — CC BY
- `stream.ogg`: “Stream” by mystiscool — CC BY
- `thunderstorm.ogg`: “Thunderstorm” by RHumphries — CC BY
- `wind.ogg`: “Wind” by felix.blume — CC0

The upstream repository README links to each original Freesound source and records the
license of every file. CC BY sounds retain their author credit here and in the application
source distribution.
